# ScarWrite - Application de Gestion Financière Premium

## À propos du projet

**ScarWrite** est une application de gestion financière premium conçue pour les entreprises internationales. Elle offre un suivi en temps réel des transactions, des transferts et des rapports financiers détaillés.

## Comment éditer ce code ?

Plusieurs méthodes sont disponibles pour éditer votre application.

**Utiliser votre IDE préféré**

Si vous souhaitez travailler localement avec votre propre IDE, vous pouvez cloner ce référentiel et pousser les modifications.

Les seules exigences sont d'avoir Node.js & npm installés - [installer avec nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

Suivez ces étapes :

```sh
# Étape 1 : Clonez le référentiel en utilisant l'URL Git du projet.
git clone <YOUR_GIT_URL>

# Étape 2 : Accédez au répertoire du projet.
cd <YOUR_PROJECT_NAME>

# Étape 3 : Installez les dépendances nécessaires.
npm i

# Étape 4 : Démarrez le serveur de développement avec rechargement automatique.
npm run dev
```

**Éditer un fichier directement sur GitHub**

- Accédez au(x) fichier(s) souhaité(s).
- Cliquez sur le bouton "Éditer" (icône de crayon) en haut à droite de la vue du fichier.
- Apportez vos modifications et validez les modifications.

**Utiliser GitHub Codespaces**

- Accédez à la page principale de votre référentiel.
- Cliquez sur le bouton "Code" (bouton vert) près du haut à droite.
- Sélectionnez l'onglet "Codespaces".
- Cliquez sur "Nouveau codespace" pour lancer un nouvel environnement Codespace.
- Éditez les fichiers directement dans Codespace et validez vos modifications une fois terminé.

## Quelles technologies sont utilisées pour ce projet ?

Ce projet est construit avec :

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
- Dexie.js (IndexedDB)
- React Router
- React Hook Form

## Fonctionnalités principales

- **Dashboard en temps réel** : Vue d'ensemble des soldes et des revenus
- **Gestion des transferts** : Suivi des transactions internationales
- **Rapports financiers** : Générer des rapports PDF détaillés
- **Calendrier interactif** : Visualiser les transactions par date
- **Gestion des produits** : Administrer l'inventaire
- **Mode hors ligne** : Accès complet même sans connexion Internet
- **Sécurité PIN** : Protection par code PIN
- **Sauvegarde et restauration** : Exporter/importer les données
